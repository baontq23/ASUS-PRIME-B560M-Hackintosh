# Features

| Feature                              | Status | Dependency          |
| :----------------------------------- | ------ | ------------------- |
| iCloud, iMessage, FaceTime           | ✅   | Whitelisted Apple ID, Valid SMBIOS  |
| AirDrop                              | ✅   | Native  |
| Audio                                  | ✅   | `AppleALC.kext` with Layout ID = 12   |
| WiFi                                 | ✅   | Native  |
| Bluetooth                            | ✅   | Native  |
| Ethernet                             | ✅   | `IntelMausi.kext`  |
| USB 2.0, USB 3.0, Type-C                   | ✅   | `USBToolBox`    |
